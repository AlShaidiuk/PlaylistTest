Страница index.html
Страница пользователя. Пользователь выбирает песню из списка и подтверждает свой выбор нажав на зеленую галочку. Голосование может происходить один раз до сброса значений на странице admin.html или при перестроении списка на странице cust.html. После голосования пользователю присваивается номер который участвует в случайном выборе поющего.
Страница admin.html
Страница, на которой отображается прогрессбар голосования для каждой песни, которая была добавлена новой строкой на странице cust.html. Голосование собирается в БД Firebase и загружается на страницу admin.html. При нажатии на кнопку "Сбросить все голоса" БД голосовавших очищаеться. Метаданные (следующий номер пользователя) сохраняются. При нажатии на кнопку "Кто будет петь?" случайно выбирается номер пользователя из самого популярного ответа, или из нескольких популярных ответов с равным количеством голосов.
